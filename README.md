# 💈 Barbershop Landing Page

Landing page moderna para barbearia com integração direta para agendamento via WhatsApp.

Este projeto foi desenvolvido como parte do meu portfólio com o objetivo de demonstrar habilidades em desenvolvimento de interfaces web modernas, responsivas e focadas em conversão.

---

# 📌 Visão Geral

A Barbershop Landing Page foi projetada para barbearias que desejam ter uma presença digital simples, elegante e eficiente.

A página permite apresentar serviços, mostrar cortes realizados e facilitar o agendamento de clientes através do WhatsApp.

Principais objetivos do projeto:

- Criar uma presença online profissional para barbearias
- Apresentar serviços de forma clara e visual
- Exibir portfólio de cortes
- Facilitar o contato e agendamento de clientes
- Oferecer uma experiência moderna e responsiva
